# Sound-Reacted-Magnetic-Fluid

This is a project for the Magnetic fluid microphone

1. Experimental Objectives and System Overview
The goal of this project is to design and produce a ferrofluid-based visual interactive device that changes dynamically with the rhythm of music. This project integrates multiple disciplines, including acoustic signal processing, microcontroller algorithm application, and electromagnetic physics.
